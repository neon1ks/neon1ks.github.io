Функция fmod

#include <math.h>
float fmodf(float a, float b);
double fmod(double a, double b);
long double fmodl(long double a, long double b);

Функции fmodf() и fmodl() определены в версии С99.

Каждая функция семейства fmod() возвращает остаток от деления аргументов а/b.

Пример

Данная программа выводит на экран число 

1.0

являющееся остатком деления 10/3. 

#include <math.h>
#include <stdio.h>

int main(void)
{
  printf("%1.1f", fmod(10.0, 3.0));

  return 0;
}
