Функция remquo

#include <math.h>
float remquof(float a, float b, int *quo);
double remquo(double a, double b, int *quo);
long double remquol(long double a, long double b, int *quo);

Функции remquo(), remquof() и remquol() определены в версии С99.

Каждая функция семейства remquo() возвращает остаток от деления значений аргументов а/b. При этом целое, адресуемое параметром quo, будет содержать частное.
