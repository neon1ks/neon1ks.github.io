Функция fma

#include <math.h>
float fmaf(float a, float b, float с);
double fma(double a, double b, double c);
long double fmal(long double a, long double b, long double c);

Функции fma(), fmaf() и fmal() определены в версии С99

Каждая функция семейства fma() возвращает значение выражения а*b+с. Округление выполняется только один раз, после завершения всей операции.

Зависимые функции
