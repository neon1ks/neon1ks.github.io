Функция ldexp

#include <math.h>
float ldexpf(float num, int exp);
double ldexp(double num, int exp);
long double ldexpl(long double num, int exp);

Функции ldexpf() и ldexpl() добавлены в версии С99.

Каждая функция семейства ldexp() возвращает значение выражения num * 2exp.

Пример

Данная программа выводит число 

4
#include <math.h>
#include <stdio.h>
int main(void)
{
  printf("%f", ldexp(1,2));
  return 0;
}
