Функция labs

#include <stdlib.h>
long labs(long num);

Функция labs() возвращает абсолютное значение аргумента num.

Пример

Приведенная ниже функция преобразует введенное с клавиатуры число в его абсолютное значение: 

long int get_labs()
{
  char num[80];

  gets(num);

  return labs(atol(num));
}
