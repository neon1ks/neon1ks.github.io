Функция abs

#include <stdlib.h>
int abs(int num);

Функция abs() возвращает абсолютное значение целочисленного аргумента num.

Пример

Эта функция преобразует введенное пользователем число в его абсолютное значение: 

int get_abs(void)
{
  char num[80];

  gets(num);
  return abs(atoi(num));
}
