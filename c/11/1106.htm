Однострочные комментарии

Благодаря Стандарту С99, в языке С появились однострочные комментарии. Комментарий такого вида начинается с // и доходит до конца строки. Например, 

// Это комментрий
int i; // это другой комментарий

Однострочные комментарии также поддерживаются языком C++. Они удобны тогда, когда нужны краткие замечания, помещающиеся в одной строке. Многие программисты для длинных описаний используют традиционные для языка С многострочные комментарии, оставляя однострочные комментарии только для объяснений, нужных "по ходу дела".
